ChartABull

Simple charts for a simple man.
These aren't supposed to detail all of the data, just present an overview.

Line | Donut | Bar | Github

The Line Chart

Takes an array of values and graphs them in the simplest way possible. The last value is the value shown in the graph.

Size, fonts, and colors can be customized.

Send me lots of numbers.

The Simple Donut Chart

The donut chart only shows one value as a percentage of the whole.



The Bar Chart

Takes an array of items and just creates bars to fit the size of your canvas.
